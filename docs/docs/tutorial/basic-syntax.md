# 基本语法

## 注释

Oraset 支持两种类型的注释：

### 单行注释

使用 `!!` 开始单行注释：

```oraset
!! 这是一个单行注释
name = `John`; !! 这也是一个注释
```

### 多行注释

使用 `!*` 开始，`*!` 结束多行注释：

```oraset
!* 这是一个
多行注释
可以跨越多行 *!
age = 25;
```

## 变量

### 变量声明与赋值

在 Oraset 中，变量不需要显式声明类型，直接赋值即可：

```oraset
name = `John`;      !! 字符串
age = 25;           !! 整数
price = 9.99;       !! 浮点数
is_student = true;  !! 布尔值
```

### 变量命名规则

- 变量名只能包含字母、数字和下划线
- 变量名不能以数字开头
- 变量名区分大小写
- 变量名不能使用关键字（如 `if`、`else`、`def` 等）

## 数据类型

Oraset 支持以下数据类型：

- **字符串**：使用反引号 `` ` `` 包围
- **数字**：整数和浮点数
- **布尔值**：`true` 和 `false`
- **数组**：使用 `arr()` 函数创建
- **对象**：使用 `obj()` 函数创建

## 操作符

### 算术操作符

| 操作符 | 描述 | 示例 |
|-------|------|------|
| `+`   | 加法 | `1 + 2` |
| `-`   | 减法 | `5 - 3` |
| `*`   | 乘法 | `2 * 4` |
| `/`   | 除法 | `10 / 2` |
| `%`   | 取余 | `7 % 3` |
| `**`  | 幂运算 | `2 ** 3` |

### 比较操作符

| 操作符 | 描述 | 示例 |
|-------|------|------|
| `==`  | 等于 | `x == y` |
| `!=`  | 不等于 | `x != y` |
| `<`   | 小于 | `x < y` |
| `>`   | 大于 | `x > y` |
| `<=`  | 小于等于 | `x <= y` |
| `>=`  | 大于等于 | `x >= y` |

### 逻辑操作符

| 操作符 | 描述 | 示例 |
|-------|------|------|
| `&&`  | 逻辑与 | `x && y` |
| `||`  | 逻辑或 | `x || y` |
| `!`   | 逻辑非 | `!x` |

### 赋值操作符

| 操作符 | 描述 | 示例 |
|-------|------|------|
| `=`   | 赋值 | `x = 5` |
| `+=`  | 加赋值 | `x += 2` |
| `-=`  | 减赋值 | `x -= 2` |
| `*=`  | 乘赋值 | `x *= 2` |
| `/=`  | 除赋值 | `x /= 2` |

## 表达式

表达式是由操作符和操作数组成的组合，用于计算值：

```oraset
result = (10 + 5) * 2; !! 结果为 30
area = 3.14 * (radius ** 2); !! 计算圆的面积
```

## 语句

每条语句以分号 `;` 结束：

```oraset
x = 10;
y = 20;
sum = x + y;
show(`Sum: `, sum);
```

## 缩进

Oraset 对缩进没有严格要求，但为了代码可读性，建议使用 4 个空格或 1 个制表符进行缩进：

```oraset
def greet(name) {
    if (name == `John`) {
        show(`Hello, John!`);
    } else {
        show(`Hello, `, name, `!`);
    }
}
```

## 转义字符

在字符串中，可以使用反斜杠 `\` 来表示转义字符：

| 转义字符 | 描述 |
|---------|------|
| `\n`    | 换行 |
| `\t`    | 制表符 |
| `\\`    | 反斜杠 |
| `` \` `` | 反引号 |
| `\"`    | 双引号 |
| `\'`    | 单引号 |

示例：

```oraset
message = `Hello\nWorld!`; !! 包含换行
path = `C:\\Users\\John`; !! 包含反斜杠
```

## 代码风格建议

- 使用有意义的变量名
- 为复杂代码添加注释
- 保持一致的缩进风格
- 将长行分成多行以提高可读性
- 为函数和类添加文档字符串
